# PDF-ZIP合并工具

一个简单的Web工具，用于将PDF和ZIP文件合并为一个文件。

## 功能

- 上传PDF文件
- 上传ZIP文件（支持ZIP、RAR、7Z、TAR、GZ格式）
- 选择输出文件后缀（.pdf 或 .zip）
- 实时显示文件大小信息
- 拖放文件上传
- 浏览器本地处理，无需上传到服务器

## 使用方法

1. 上传一个PDF文件
2. 上传一个ZIP文件
3. 输入输出文件名
4. 选择输出后缀
5. 点击"合并文件"
6. 下载合并后的文件

## 使用合并后的文件

- 保持后缀为 `.pdf` - 作为PDF文件打开
- 更改后缀为 `.zip` - 作为ZIP文件解压

## 技术说明

所有文件处理都在浏览器本地完成，不会上传到任何服务器。

## 部署

可直接部署到GitHub Pages或任何静态网站托管服务。
