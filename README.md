# 多文件合并工具

一个可以将多个文件（2个及以上）合并为一个文件的Web工具。通过更改文件后缀名，可以访问合并文件中的不同内容。

## 功能特性

- 支持任意数量的文件合并（最少2个）
- 支持拖放文件上传
- 可自定义输出文件名和后缀
- 支持多种文件格式：
  - 文档：PDF, DOC, DOCX, TXT, MD等
  - 图片：PNG, JPG, GIF, SVG等
  - 音频：MP3, WAV, FLAC等
  - 视频：MP4, AVI, MOV等
  - 压缩文件：ZIP, RAR, 7Z等
  - 代码文件：JS, HTML, CSS, Python等
- 实时内存使用提示
- 文件顺序拖拽调整
- 添加文件分隔符和元数据选项

## 工作原理

本工具基于以下原理：

1. **格式识别靠文件头**：大多数程序通过文件头部信息识别文件格式，而不是文件后缀
2. **拼接**：将多个文件拼接成一个文件
3. **后缀决定打开方式**：文件后缀只影响默认打开程序，不改变文件内容

## 使用方法

### 基本使用
1. 访问网站
2. 上传至少2个文件（推荐第一个文件为图片格式）
3. 调整文件顺序（拖拽文件项）
4. 设置输出文件名和后缀
5. 点击"合并文件"
6. 下载合并后的文件

### 访问不同内容
- 保持默认后缀（如.png）访问第一个文件
- 更改后缀为第二个文件类型（如.zip）访问第二个文件
- 依此类推访问其他文件
